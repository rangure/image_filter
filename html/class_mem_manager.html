Detailed Description

this class offers functionalities to manage memory. The map stores the address of the allocated memroy and the type of the allocated memory. allocate should be called when new is called, and deallocate should be called when delete is called. however, this class is meant to be used when a there's exception and you need to quit the program with allocated memory. if there's no exception between new and delete this shouldn't be used.
